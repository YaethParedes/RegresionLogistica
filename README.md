Este repositorio contiene el cuaderno de python con la aplicación del modelo de regresión logística usando el paquete Stats.
Usando la misma partición de datos se ajustan dos modelos, uno considerando la edad como variable discreta y otro con
los mismos datos de edad pero categorizados en 8 grupos.

El data CHD.csv se tomó de la tabla 1.1 del capítulo Introduction to the Logistic Regression Model,
del libro "Applied Logistic Regression" por David W. Hosmer, JR, Stanley Lemeshow y Rodney X. Sturdivant.
